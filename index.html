Applications and guides published by man-schu

Feel free to download the free open source applications and to use them on your own server.

Rašyti Touch Typing Trainer

Rašyti is a multilingual and multi-keyboard offline and online touch typing trainer.

Find the application here: https://man-schu.github.io/rasyti/. There is no need to install anything. It can be used right from Github.

A standalone desktop version based on Electron can be downloaded on https://github.com/man-schu/rasyti/releases/latest/.

The source code is hosted here: https://github.com/man-schu/rasyti/.

Dacapo Editor - Music Sheet Editor

DaCapo Editor is a music sheet editor based on the ABC syntax with which you can write easily music scores and partitions.

Find the application here: https://man-schu.github.io/dacapoeditor/. There is no need to install anything. It can be used right from Github.

A standalone desktop version based on Electron can be downloaded on https://github.com/man-schu/dacapoeditor/releases/latest/.

The source code is hosted here: https://github.com/man-schu/dacapoeditor/.

JS Recorder - Audio, Video and Screen Recorder

JSRecorder is a tool to record audio, video and the screen. It works fully in the browser with nothing being stored on the server.

Find the application here: https://man-schu.github.io/jsrecorder/. There is no need to install anything. It can be used right from Github.

A standalone desktop version based on Electron can be downloaded on https://github.com/man-schu/jsrecorder/releases/latest/.

The source code is hosted here: https://github.com/man-schu/jsrecorder/.
